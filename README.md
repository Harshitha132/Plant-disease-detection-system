# 🌱 LeafScan AI - Plant Disease Detection System

## 📌 Overview

LeafScan AI is an AI-powered Plant Disease Detection System that helps farmers and agricultural professionals identify plant diseases from leaf images. The system uses Deep Learning models built with TensorFlow and provides instant disease diagnosis through an interactive web interface.

The application allows users to upload plant leaf images, detect diseases, view confidence scores, access disease information, and interact with a voice-enabled assistant.
